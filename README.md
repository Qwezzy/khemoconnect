# KhemoConnect - 2026 Upgrade

KhemoConnect is the web & digital experience division of Khemo IT Solutions (Gauteng, South Africa, est. 2008).

This project is the new flagship marketing site and positioning for the 2026 era:

- **GEO (Generative Engine Optimization)** - Get cited and recommended by ChatGPT, Perplexity, Gemini and Google AI Overviews.
- **AI Webpages & Hyper-Personalization** - Websites that change based on who is looking.
- **Day-One AI Automation Packages** - Lead → classify → proposal → WhatsApp in minutes.
- **Own the stack** - Scalable, AI-optimized hosting as the foundation for lower cost and higher performance.

**Target audience:** Startups, Scale-ups, Local SMEs.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000

The site itself is built with the GEO and personalization techniques we sell.

## Key pages / sections
- Hero with the new pitch
- Capabilities (GEO + AI Webpages + Automation + Hosting advantage)
- Live interactive demos (Personalization + Automation pipeline)

- GEO explanation (self-referential)
